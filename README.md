# Canno

Canno is a spreadsheet program.

Engine written in C++ and GUI in Python using Tkinter.

## Features
- [X] `Formulas`
    - [X] `Cell references`
    - [X] `Binary operations`
    - [X] `Functions`
    - [ ] `Parentheses`
- [X] `Graph dependency tree`
- [ ] `Frontend updates only changed cells`
- [X] `Error handling`
- [ ] `Circular dependency detection`
- [X] `Ranges =SUM(A1:A5)`
- [ ] `Saving & loading from file`
